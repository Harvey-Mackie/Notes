# List of Commands

## Viewing and Editing
- View contents of file - `cat [file]`
- Edit contents of file - `nano [file]`
- Open in visual studio code - `code [file]`
- Output results to a new file - `[command] > [file]` e.g. `git pull > temp.txt`

## Moving
- Move folder/file - `mv [file] [destination]`

## Removing
- Remove folder/file
	- `rm [file]`
	- `rmdir [folder]`
